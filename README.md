## Features

## Getting Started

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

## Tech Stack

- React
- Typescript
- Vite
- Tailwind
- Shadcn/ui
- React Flow
- Recharts
